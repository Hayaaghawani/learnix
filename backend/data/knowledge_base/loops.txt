Loops allow repeated execution of a block of code.

C++ has three main loop types:

1. for loop - use when you know how many times to repeat
for (int i = 0; i < n; i++) {
    // body
}

2. while loop - use when you repeat based on a condition
while (condition) {
    // body
}

3. do-while loop - like while but executes the body at least once
do {
    // body
} while (condition);

Common mistake: infinite loops happen when the condition never becomes false.

Loop control:
- break: exits the loop immediately
- continue: skips the rest of the current iteration

Example - sum of 1 to n:
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum += i;
}
