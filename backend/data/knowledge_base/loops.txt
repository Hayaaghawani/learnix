# Loops in Programming

## What is a Loop?
A loop is a control structure that repeats a block of code while a condition is true.

## Types of Loops

### 1. For Loop
Used when you know the number of iterations in advance.
```
for i in range(5):
    print(i)  # Prints 0 to 4
```

### 2. While Loop
Used when the condition is checked before each iteration.
```
count = 0
while count < 5:
    print(count)
    count += 1
```

### 3. Do-While Loop
Used when you need at least one iteration (not available in Python).

## Common Patterns
- **Counter**: for i in range(n)
- **Condition**: while condition_is_true()
- **Collection**: for item in collection
- **Nested**: loops within loops

## Performance Considerations
- Minimize operations inside the loop
- Avoid nested loops when possible
- Break/continue statements can optimize performance

## Break and Continue
- **break**: Exits the loop immediately
- **continue**: Skips to the next iteration
