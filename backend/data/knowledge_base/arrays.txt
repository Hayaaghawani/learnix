# Arrays in Programming

## What is an Array?
An array is a collection of elements of the same data type stored in contiguous memory locations.

## Key Concepts
- **Index**: The position of an element in the array (usually starts at 0)
- **Length**: The total number of elements in the array
- **Bounds**: The valid range of indices (0 to length-1)

## Common Operations
1. **Access**: array[index] - O(1)
2. **Insert**: Adding an element - O(n)
3. **Delete**: Removing an element - O(n)
4. **Search**: Finding an element - O(n)

## Example
```
array = [1, 2, 3, 4, 5]
array[0] = 1   # First element
array[4] = 5   # Last element
array[5] = Error (out of bounds)
```

## Time Complexity
- Access: O(1)
- Search: O(n)
- Insert: O(n)
- Delete: O(n)
